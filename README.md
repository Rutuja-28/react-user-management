# React User Management Application

A React CRUD application built using React Hooks, React Router, Axios, Tailwind CSS, and DummyJSON API.

## Features

- View Users
- Add User
- Edit User
- Delete User
- Search Users
- Filter Users
- Pagination
- Loading States
- Form Validation
- Responsive Design

## Technologies

- React
- React Router DOM
- Axios
- Tailwind CSS
- React Hot Toast
- DummyJSON API

## Installation

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
